HTML/CSS and JavaScript Projects

About GITA 2

GITA 2 is the second level of the Global IT Academy program at Brea Olinda High School. I took this class when I was a junior. GITA 2 aims to teach students the languages of HTML/CSS and JavaScript, which are used for websites. In GITA 2, we coded our projects using Notepad++, which can be downloaded here, but a Windows computer is necessary for the program. My projects are listed below, starting with my final project and continuing in reverse chronological order. These projects will open directly in-browser and do not require a download. Enjoy!

To view projects: All platforms can view these projects. Clicking on each link will open up a new tab that displays the project.

This is my final project, which is a rendition of the classic arcade game "Galaga" by Namco. Space Invaders (.html) Completed: 2/3/16

This project is an extension of the mini space invaders project. It uses arrays to keep track of 5 invaders. The user must hit each invader 3 times each to win before the invaders hit the spaceship. This taught me how to use arrays in JavaScript to keep track of multiple objects and values.

Multiple Ball Bounce (.html) Completed: 2/2/16

This project uses arrays to keep track of multiple balls and have them bounce around the screen. The balls also bounce off of each other as well as the walls. This taught me how to use arrays to create multiple objects and control their movements while also implementing collisions.

Mini Space Invaders (.html) Completed: 1/19/16

This is a mini project to make a space invaders game with only one alien. The alien and the ship are drawn through the canvas tag, and the alien shrinks as it is hit. The alien can sustain three hits before it dies and the user wins. If the alien survives, the user loses. This taught me how to make a basic game and keep control of an object that can detect collisions.
