GITA 4 Projects

About GITA 4

AP Computer Science (GITA 4) is the final level of the Global IT Academy program. I took this class as a senior in high school. GITA 4 aims to teach students Java, which is the language that is tested on the AP Computer Science test. Java is used for many phone and computer applications and plugins. AP Computer Science teaches computer science techniques, like object-oriented programming and data structures, which makes code more efficient and easier to manage. I coded my projects using Eclipse, which can be downloaded here. My projects are listed below, starting with my final project and continuing in reverse chronological order. Windows may flag the program as malware even though it is safe, so do not be alarmed when your browser may try to block it from downloading. Enjoy!

To view the projects: All platforms can view these projects, but Java must be downloaded. Java is necessary to view my projects. Download it here. Clicking on each link will download a *.jar file, which can be run once Java is downloaded and installed.

Final Project

Crunch Time (.zip)

Completed 6/11/17

I chose to make my own game for my final project, which I named Crunch Time. Crunch Time is an arcade survival game where the user tries to survive waves of enemies that increasingly grow stronger. I used the techniques that I learned in AP Computer Science/GITA 4 to make my code object-oriented and as efficient as possible. This game also uses some trigonometry and physics to control the bullets and speed of the objects. I'm proud to have been able to come up with an idea for my own game and to implement it successfully. This project comes as a *.zip file. Download the file and extract it to reveal the *.jar file, which can be run. Enjoy!
