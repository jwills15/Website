GITA 1 Projects

About GITA 1

GITA 1 is the first level of the Global IT Academy program. I took this class when I was a sophomore in high school. GITA 1 aims to teach students the language of C#, which is used for Windows applications. GITA 1 teaches how to use Visual Studio, which was created by Microsoft to easily create C# programs. Visual Studio can be downloaded here, but a Windows computer is necessary for the program. My projects are listed below, starting with my final project and continuing in reverse chronological order. My projects require a Windows computer to view. Windows may flag the program as malware even though it is safe, so do not be alarmed when your browser may try to block it from downloading. Enjoy!

To view the projects: Only windows computers can view these projects. The download for each project is a *.zip file, which needs to be extracted before it can run. Once extracted, it will be a *.exe file.

Galaga (.zip)

I chose to make Galaga for my final project. Galaga was my favorite game as a child and I always loved playing it at the arcade.
